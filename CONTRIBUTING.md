# Участие в разработке MCP-1C

Мы рады вашему участию в развитии проекта!

## Как внести вклад

### 1. Сообщения об ошибках

- Используйте GitHub Issues
- Опишите шаги воспроизведения
- Укажите версию Python и ОС
- Приложите логи (если есть)

### 2. Предложения по улучшению

- Создайте Issue с пометкой `enhancement`
- Опишите предлагаемую функциональность
- Объясните почему это полезно

### 3. Pull Requests

1. Форкните репозиторий
2. Создайте ветку: `git checkout -b feature/my-feature`
3. Внесите изменения
4. Напишите тесты
5. Запустите проверки:
   ```bash
   # Линтинг
   ruff check src tests

   # Форматирование
   black src tests

   # Тесты
   pytest
   ```
6. Создайте Pull Request

## Стиль кода

- Python 3.11+
- Форматирование: Black
- Линтинг: Ruff
- Type hints обязательны
- Docstrings для публичных методов

## Структура коммитов

```
feat: добавить новый инструмент X
fix: исправить ошибку в Y
docs: обновить документацию
test: добавить тесты для Z
refactor: рефакторинг модуля W
```

## Тестирование

```bash
# Запуск тестов
pytest

# С покрытием
pytest --cov=src/mcp_1c

# Только конкретный файл
pytest tests/unit/test_metadata_tools.py -v
```
