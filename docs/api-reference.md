# API Reference

## Metadata Tools

### metadata.init

Инициализация индекса метаданных конфигурации.

**Параметры:**

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| `path` | string | Да | Путь к корню конфигурации |
| `full_reindex` | boolean | Нет | Полная переиндексация (по умолчанию: false) |
| `watch` | boolean | Нет | Отслеживать изменения (по умолчанию: true) |

**Возвращает:** Статистику индексации

```json
{
  "total": 150,
  "updated": 150,
  "skipped": 0,
  "errors": 0
}
```

---

### metadata.list

Получить список объектов по типу.

**Параметры:**

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| `type` | string | Да | Тип объекта метаданных |

**Допустимые типы:**
- `Catalog` — Справочники
- `Document` — Документы
- `Report` — Отчёты
- `DataProcessor` — Обработки
- `InformationRegister` — Регистры сведений
- `AccumulationRegister` — Регистры накопления
- `AccountingRegister` — Регистры бухгалтерии
- `CalculationRegister` — Регистры расчёта
- `Enum` — Перечисления
- `CommonModule` — Общие модули
- `ChartOfCharacteristicTypes` — Планы видов характеристик
- `ChartOfAccounts` — Планы счетов

**Возвращает:** Список объектов

```json
{
  "type": "Catalog",
  "count": 25,
  "objects": [
    {"name": "Номенклатура", "synonym": "Номенклатура"},
    {"name": "Контрагенты", "synonym": "Контрагенты"}
  ]
}
```
